П1.1. Анализ температурной нестабильности режима  работы
биполярного транзистора


Ток коллектора в схеме с ОЭ существенно зависит от температуры. Основных причин две. Первая – зависимость от температуры тока эмиттера:

IЭ= IЭ0 ( exp {e UБЭ / kT} - 1 ),

где e - заряд электрона, – напряжение на эмиттерном переходе, k - постоянная Больцмана, T - абсолютная температура; kT/e = φТ – температурный потенциал. Ток IЭ0 – обратный ток эмиттерного перехода – сильно зависит от температуры:

IЭ0 = IНАС exp { -E /φТ }.

Рис.П1.1

Здесь IНАС –  ток насыщения, который протекал бы через переход при нулевом потенциальном барьере; E – ширина запрещенной зоны. Повышение температуры приводит к росту тока эмиттера, а следовательно, и тока коллектора. Это видно из рис.П1.1, если зафиксировать UБЭ. Ток коллектора останется неизменным, если с ростом температуры уменьшать UБЭ, что также видно из рисунка. Для практики важно, что рост температуры приводит к параллельному сдвигу вольт-амперных характеристик влево. Это дает возможность учесть температурную нестабильность с помощью коэффициента температурного сдвига характеристик:

γТ =∂UБЭ / ∂T ≈ (E – UБЭ ) / T = – (2...3 ) мВ / град.

Таким образом, изменение температуры эмиттерного перехода эквивалентно изменению напряжения смещения на базе ΔUБ т = γТ ΔT.  Это позволяет  условно ввести в цепь базы эквивалентный генератор напряжения ΔUБ т, учитывающий температурную нестабильность тока коллектора.

Вторая причина температурной нестабильности – влияние температурных изменений обратного тока коллектора IК0. Они невелики по абсолютной величине, однако, протекая в цепи базы, дают изменение падения напряжения на резисторах в базовой цепи, которое затем усиливается транзистором. Приращение тока IК0 составляет 2...2,5 раза на каждые 10 градусов. В справочниках приводится значение тока IК0 при некоторой температуре T0. Значение тока при другой температуре T можно найти по формуле:

IК0 (T ) = IК0 ( T0 )( exp{a ( T - T0 )} - 1),

 где a @ 0,06 для кремниевых транзисторов.

Температурная нестабильность токов транзистора позволяет говорить об их приращениях как о переменных токах, изменяющихся с очень низкой частотой. По этой причине для анализа процессов может быть использована схема замещения транзистора с использованием низкочастотных Y-параметров. Удобно использовать схему замещения рис.2.8,б, которая справедлива как для полевого, так и для биполярного транзистора, причем для последнего на низкой частоте   Y21 = S, с учетом (2.3)

YВХ = gВХ = g11н+ K0 g12 @ g11Н

(П1.1)

- ввиду малости g12. Кроме того, Y22 @ g22. Полученная с учетом этого схема замещения транзистора показана на рис.П1.2. В ней также учтены два источника температурной нестабильности в виде генератора напряжения ΔUБт (первая причина), включенного в базовой цепи, и генератора тока ΔIК0 (вторая причина), включенного между базой и коллектором. Резистор RВН отражает общее сопротивление внешних резисторов,  подключенных к базе, по переменному току.

Рис.П1.2

Температурную нестабильность коллекторного тока отражает генератор тока ΔIК = SΔUБ, где (с учетом нулевого внутреннего сопротивления генератора напряжения ΔUБ т и нулевой внутренней проводимости ГТ ΔIК0)

 Следовательно,

При RВН → ∞ (близко к случаю смещения фиксированным током базы) влияние ΔUБ т минимально, т.е. ΔIК1 = 0, зато ΔIК2 = ΔIК2max  и

ΔIК =  ΔIК2max  = S  ΔIК0 / g11 = β0  ΔIК0.

 Наоборот, при RВН → 0 (близко к случаю смещения фиксированным напряжением на базе) минимально влияние  ΔIК0, ΔIК2 = 0, а ΔIК1 = ΔIК1max  и ΔIК = ΔIК1max =SΔUБ т. Вообще говоря, обе рассмотренные простейшие схемы с точки зрения температурной стабильности малоэффективны.

Способность схемы обеспечивать температурную стабильность режима оценивается коэффициентами стабилизации SТ1, SТ2, при этом

ΔIК =  ΔIК1 + ΔIК2 = SТ1  ΔIК1max+ SТ2  ΔIК2max,

т.е.

SТ1 = ΔIК1 / ΔIК1max;  SТ2 =  ΔIК 2  / ΔIК 2max.

 Рассчитав значения SТ1, SТ2 для конкретной схемы температурной стабилизации, далее можно найти общую температурную нестабильность тока коллектора:

ΔIК = S ΔUБ т SТ1 + β0  ΔIК0 SТ2.
